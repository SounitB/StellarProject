# 🌱 Carbon Tracker (Soroban Smart Contract)

## 📌 Project Description

Carbon Tracker is a simple decentralized application built on Stellar’s Soroban smart contract platform. It allows users to track their carbon emissions in a transparent and immutable way on the blockchain.

The goal is to promote environmental awareness by giving individuals and organizations a tool to monitor their carbon footprint.

---

## ⚙️ What It Does

This smart contract enables:

* Recording carbon emissions per user
* Storing emission data on-chain
* Retrieving total emissions for any user
* Resetting emission records when needed

Each user is identified by a unique string (e.g., wallet ID, username, or organization name), and their emissions are stored cumulatively.

---

## ✨ Features

### 🔹 Add Emissions

Users can log carbon emissions (e.g., from travel, energy use, etc.), which are added to their total.

### 🔹 View Emissions

Anyone can query the total carbon footprint of a specific user.

### 🔹 Reset Emissions

Users can reset their emission data (useful for monthly or yearly tracking cycles).

### 🔹 On-Chain Transparency

All data is stored on the Stellar blockchain, ensuring:

* Transparency
* Immutability
* Trustless verification

### 🔹 Lightweight & Efficient

* Built using Soroban SDK
* Minimal storage usage
* Simple and extendable logic
